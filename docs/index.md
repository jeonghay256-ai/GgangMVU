# MVU Zod 변수 카드 직접 만들기

### 📺 먼저 영상 시청하기
**AI를 사용하지 않고, 변수와 EJS(임베디드 자바스크립트)만으로 제어되는 프롬프트를 어떻게 작성할까요?**
먼저 전체적인 흐름을 파악한 뒤, 문서를 읽으며 세부 사항을 이해해 봅시다.
*(영상에서 언급되는 MVU beta를 모르셔도 괜찮습니다!)*

---

## 📖 문서 소개

* **기존 사용자라면:** 이미 MVU나 MVU beta 사용법을 알고 계시다면, **[[부록] 구형 MVU 카드 이전]** 문서를 통해 MVU Zod가 이전 버전보다 얼마나 더 간단하고 강력해졌는지 확인하실 수 있습니다.
* **처음 오셨다면:** MVU나 변수 카드를 전혀 접해본 적이 없다면, *"왜 이걸 써야 하지?"* 하고 궁금하실 텐데요...
    가장 핵심적인 이유는, 캐릭터 카드를 변수화하면 **마치 게임처럼 상호작용**할 수 있고, **토큰을 절약**하며, **이야기 전개에 따라 프롬프트를 동적으로 변경**할 수 있기 때문입니다!
