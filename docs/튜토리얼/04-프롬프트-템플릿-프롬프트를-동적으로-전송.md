# 프롬프트 템플릿: 프롬프트를 동적으로 전송

상황에 맞게 자동으로 프롬프트를 생성하고 전송하는 방법을 알아봅니다.

## 동적 프롬프트란?

동적 프롬프트는 현재 상태, 변수 값, 조건 등에 따라 자동으로 생성되는 프롬프트입니다. 매번 수동으로 작성할 필요 없이 상황에 맞는 프롬프트가 자동으로 구성됩니다.

## 설정 방법

### 1. 트리거 조건 설정

```javascript
{
  "trigger": {
    "event": "변수변경",
    "target": "상태"
  }
}
```

### 2. 프롬프트 템플릿 정의

```javascript
{
  "template": "{{상태}}일 때 {{캐릭터}}의 반응을 작성해주세요."
}
```

### 3. 전송 설정

```javascript
{
  "autoSend": true,
  "delay": 0
}
```

## 활용 예시

### 상태 기반 프롬프트

```javascript
{
  "conditions": [
    {
      "if": "상태 === '전투'",
      "prompt": "전투 상황을 묘사해주세요."
    },
    {
      "if": "상태 === '대화'",
      "prompt": "대화 장면을 진행해주세요."
    }
  ]
}
```

!!! tip "최적화 팁"
    자주 사용하는 동적 프롬프트는 템플릿으로 저장해두면 편리합니다.

## 주의사항

!!! warning "주의"
    너무 많은 동적 프롬프트를 동시에 활성화하면 성능에 영향을 줄 수 있습니다.
