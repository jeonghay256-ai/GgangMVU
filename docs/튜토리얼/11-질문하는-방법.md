# 부록. 질문하는 방법

튜토리얼을 따라 작성하던 중 문제가 발생하면, 포스트([류뇌(Classbrain)](https://discord.com/channels/1134557553011998840/1400321862235328552/1400321862235328552)/[여정(Journey)](https://discord.com/channels/1291925535324110879/1400321714298294272/1400321714298294272))에 방문하여 질문해 주세요!

## 질문 전 준수 사항

* 질문하기 전에, 튜토리얼의 처음부터 문제가 발생한 부분까지 다시 한번 읽어보면서 조작에 실수가 있었는지, 혹은 튜토리얼에서 이미 수정 방법을 안내했는데 놓친 것은 아닌지 확인해 주세요! 포스트에 올라오는 대부분의 질문은 이러한 확인으로 해결될 수 있습니다.
* 질문할 때 스크린샷이 필요할 수 있습니다. **반드시 전체 화면을 캡처해 주세요!** 일부분만 잘라서 올리지 마세요! 어디가 문제인지 모르기 때문에 질문을 하는 것인데, 만약 본인이 문제라고 생각하는 부분만 잘라서 올린다면, 사실은 다른 곳에 문제가 있었을 경우 어떻게 해결할 수 있을까요? 일부분만 캡처하는 것은 문제 해결을 더 어렵게 만들 뿐입니다!!!

## 제공해야 할 정보

튜토리얼을 다시 읽어보았음에도 여전히 문제를 해결할 수 없다면, 포스트를 통해 저희에게 질문해 주세요. 질문할 때는 가능한 한 다음 정보를 포함해 주세요:

* **프롬프트 템플릿(Prompt Template)**이 활성화 상태인지, **태번 헬퍼(Tavern Helper) - 메인 인터페이스 - 태번 헬퍼 매크로 비활성화**가 비활성화 상태인지 확인하셨나요?
* 관련 없는 플러그인, 전역 스크립트, 전역 정규식을 비활성화하여 그것들의 영향을 배제해 보셨나요?
* PC라면 `F12` - 콘솔(Console)에 어떤 빨간색 오류가 뜨나요? 모바일이라면 로그 뷰어에 어떤 빨간색 오류가 뜨나요?
* 작성한 변수 구조 스크립트는 어떤 형태인가요?
* 변수 구조 스크립트에 `{{user}}` 등의 매크로가 변수명으로 사용되었나요? AI가 `[substitudeMacros('{{user}}')]`를 사용하여 매크로를 변수명으로 사용할 수는 있지만, `주인공` 같이 매크로가 필요 없는 변수명으로 바꾸는 것을 권장합니다.
* 월드인포의 initvar, 변수 업데이트 규칙, 변수 목록, 변수 출력 형식 항목은 어떻게 내용을 작성하고 설정하셨나요?
* 월드인포에 토큰을 대량으로 차지하는 다른 항목이 있나요? 그 항목들은 어떻게 내용을 작성하고 설정하셨나요?
* 월드인포에 변수 업데이트 `<UpdateVariable>` 외에 다른 출력 형식이 있나요? 내용은 어떻게 작성하고 설정하셨나요?
* 채팅창 좌측 하단 요술봉 - 프롬프트 뷰어(Prompt Viewer)를 열었을 때, 현재 토큰은 얼마인가요? 어떤 모델을 사용 중인가요? 어떤 프리셋을 사용 중인가요?
* 프롬프트 뷰어 내에서 전송되어야 한다고 생각하는 특정 프롬프트가 검색되나요?
* 채팅창 좌측 하단 요술봉 - 변수 관리자 - 메시지 층계(Message Floor)를 열었을 때, 최신 층계의 변수 상황은 어떤가요?
* 캐릭터 카드의 태번 헬퍼 캐릭터 스크립트, 지역 정규식에는 어떤 것들이 있나요?
* 만약 문제가 채팅 메시지와 관련이 있다면, 오류가 발생한 채팅 메시지는 무엇인가요?

또한, 가능하다면 캐릭터 카드 파일과 채팅 메시지를 직접 보내주시고, 어떤 조작을 했을 때 해당 오류가 다시 발생하는지 알려주시면 좋습니다.
